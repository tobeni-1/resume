# 涂雨晴个人简历网站 · 项目指引

## 项目简介
为涂雨晴打造紫色科技风个人简历网站，展示个人信息、教育/工作经历、项目经历、专题研究，支持 PDF 下载。免费部署至 GitHub Pages。

## 工作说明

### 基本规则
- **渐进开发**：每次只完成一个步骤，验证通过后再进入下一步
- **先讨论后执行**：涉及内容呈现（尤其是项目卡片详情）的决策，先与用户沟通再动手
- **保持稳定**：每步完成后确认页面可正常打开、无报错
- **记录日志**：每次开发后更新 `dev-log/` 下的当日日志文件

### 内容脱敏
- 公司名称：使用「某地铁集团」「某城投平台」等替代
- 财务数据：具体数字改为范围描述或趋势描述
- 脱敏后需用户审核确认

### 用户沟通偏好
- 用户为编程新手，需要用通俗语言解释技术决策
- 项目展示是核心竞争力的体现，需要精细打磨
- 所有内容修改需用户确认后方可定稿
