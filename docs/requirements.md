# 开发需求文档

## 项目背景
涂雨晴（2年商业分析经验）求职中，需要个人简历网站提升竞争力。

## 功能需求

### FR-1：个人信息区（Hero）
- 展示：姓名、照片、电话、邮箱、学历、求职意向
- 核心能力标签（商业分析 / 战略咨询 / 经营策划 / 财务建模等）
- 2-3句个人简介
- 3条核心优势（业务洞察 / 数据处理 / 跨部门协同）

### FR-2：时间轴（教育 & 工作经历）
- 左右双栏布局：左侧教育、右侧工作
- 教育：东北财经大学（硕士）、江西理工大学（本科）
- 工作：广东中大管理咨询集团（全职）、北京研师亦友（实习）
- 中间竖线连接，节点发光动画

### FR-3：项目经历（卡片 + 弹窗）
- 卡片正面：项目配图 + 一句话概述
- 单击卡片：弹出详情弹窗（Modal）
- 项目清单：
  1. 某地铁集团十五五战略规划
  2. 某城投平台十五五战略规划
- 弹窗内容需脱敏（公司名 + 财务数据模糊化）
- 弹窗内容需符合甲方用人部门阅读习惯

### FR-4：专题研究（卡片 + 弹窗）
- 与项目卡片相同的交互模式
- 研究清单：
  1. 世界一流轨交企业可持续发展案例研究（聚焦巴黎地铁）
  2. 国有企业发展资源循环利用产业研究

### FR-5：PDF 下载
- 下载按钮，点击直接下载 PDF 版本简历
- 文件路径：`涂雨晴简历-经营&商业分析岗.pdf`

### FR-6：导航栏
- 固定在页面顶部
- 平滑滚动到各区域
- 移动端汉堡菜单

## 非功能需求

### NFR-1：视觉风格
- 紫色主色调 + 深色背景
- 简约科技感（毛玻璃卡片、发光线条、粒子背景）
- 参考 claude.com 的排版节奏，但颜色体系不同

### NFR-2：响应式
- 桌面端（>1024px）：完整布局
- 平板端（768-1024px）：调整间距
- 手机端（<768px）：时间轴切换为单列、卡片全宽

### NFR-3：性能
- 首屏加载 < 3 秒
- 无外部框架依赖（纯 HTML/CSS/JS）
- 图标库 CDN 异步加载

### NFR-4：浏览器兼容
- Chrome / Edge / Firefox / Safari 最新两个版本
